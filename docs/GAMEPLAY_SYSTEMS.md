# Gameplay Systems Design

Concrete numbers, formulas, and mechanics definitions for every system in the game. The reference document for implementation.

## Covering:

- **Scanner System** — Discovery Sweep range per level, scan time, Integrity cost, Target Analysis detail tiers, Deep Scan duration and risk. Upgrade costs for each of the 6 Scanner Array upgrades.

- **Extractor Mechanics** — Base damage per click, damage scaling with Cutter Arrays upgrades, section HP values per wreck tier, stress accumulation formula (stress per click × number of active extractors), catastrophic failure thresholds, failure recovery penalties.

- **Probe Automation** — Base trip time, yield per trip, scaling with Probe Command and Probe Efficiency upgrades, priority queuing mechanics, probe capacity limits per upgrade level.

- **Ship Systems Repair** — Exact repair costs per percentage point for all 6 systems (Hull, Life Support, Propulsion, Maw Core, Scanner Array, Power Grid). Resource yield rates from each wreck section type. Conversion formulas.

- **Region Progression** — How many wrecks per region, Integrity cost to travel between regions, unlock conditions per region (system % thresholds), region-specific modifiers (increased stress in core-ward regions, better loot in dangerous regions).

- **Threat Encounters** — Whaler spawn frequency and detection mechanics, Ghost Signal encounter triggers, Scavenger behavior patterns, Frenzy/wreck stress events.

- **Tech Tree** — Full upgrade tree with all costs, prerequisites, and level caps. How overclocking (Exotic Alloys) interacts with each system.

- **Prestige / Endgame Loop** — If any, how Graal Spores enable new-game-plus mechanics or ascension paths.
